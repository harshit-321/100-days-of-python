# 🔐 Random Password Generator (Python)

A simple Python script to generate strong random passwords using letters, numbers, and special characters.

---

## 🚀 Features
- Generates random passwords of any length  
- Includes uppercase, lowercase, digits, and symbols  
- Beginner-friendly Python project  
- Runs in terminal/command prompt
